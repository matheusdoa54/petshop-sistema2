# Sistema Web - Pet Shop AuMiau

## Descrição
O **AuMiau** é um sistema web desenvolvido como projeto acadêmico, com o objetivo de apresentar de forma organizada os produtos e serviços oferecidos por um pet shop. A aplicação foi construída utilizando **HTML**, **CSS**, **Bootstrap** e JavaScript apenas para elementos dinâmicos.

O sistema proporciona uma navegação simples e intuitiva, permitindo que usuários visualizem informações sobre produtos e serviços, além de realizar cadastros para agendamento de serviços ou compras.

---

## Funcionalidades

### Navegação
- Menu de navegação presente em todas as páginas, com os seguintes links:
  - **Início**: leva para a página principal do sistema.
  - **Produtos**: apresenta os produtos disponíveis para compra.
  - **Serviços**: mostra os serviços oferecidos, como banho, tosa e consulta veterinária.
  - **Cadastro**: permite que clientes realizem seu cadastro e agendem serviços.
  - **Ajuda**: fornece informações sobre o sistema, suas funcionalidades e tecnologias utilizadas.

### Página Inicial
- Apresenta uma imagem principal (banner) do pet shop.
- Seção com destaques de produtos e serviços oferecidos.
- Layout responsivo, compatível com dispositivos móveis e desktop.

### Produtos
- Exibe diferentes categorias de produtos, como rações, brinquedos e acessórios.
- Cada produto possui imagem, descrição e preço.
- Ao clicar em qualquer produto, o usuário é redirecionado para a página de cadastro.

### Serviços
- Apresenta os serviços disponíveis, incluindo banho, tosa e consulta veterinária.
- Cada serviço é apresentado com imagem, descrição detalhada e preço.
- Ao clicar em qualquer serviço, o usuário é redirecionado para a página de cadastro para agendamento.

### Cadastro
- Formulário completo para coleta de informações do cliente e do pet.
- Campos incluem: nome, endereço, CPF, sexo, telefone, e-mail, nome do pet, raça, idade, data e hora do agendamento, e opção de atendimento.
- Botões de **Enviar** e **Limpar Formulário**.
- Redirecionamento para a tela inicial ao clicar no logo **AuMiau** no topo da página.

### Ajuda
- Página informativa com explicações sobre o sistema e suas funcionalidades.
- Seções detalhadas sobre o sistema, funcionalidades e tecnologias utilizadas.
- Botão **Cadastro** disponível no menu para fácil acesso.

### Layout e Estilo
- Uso de **Bootstrap 5** para responsividade e estilização rápida.
- Design moderno, com cores suaves e foco na experiência do usuário.
- Cards interativos com efeito de hover em produtos e serviços.

---

## Tecnologias Utilizadas
- **HTML5**: estrutura das páginas.
- **CSS3**: estilização customizada e responsiva.
- **Bootstrap 5**: componentes e grid responsivo.
- **JavaScript**: navegação dinâmica (cliques em produtos/serviços e redirecionamentos).
- **Imagens**: utilizadas para produtos, serviços e banner principal.
