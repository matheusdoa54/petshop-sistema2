Sobre o Sistema

Este sistema web foi desenvolvido como projeto acadêmico para apresentar os produtos e serviços de um pet shop.

Funcionalidades

  • Página inicial com apresentação do pet shop
  • Página de produtos com categorias
  • Página de serviços oferecidos
  • Navegação entre páginas através do menu

Tecnologia Utilizada

O sistema foi desenvolvido utilizando HTML, CSS e Bootstrap, com JavaScript apenas para elementos dinâmicos, conforme as regras da atividade acadêmica.
